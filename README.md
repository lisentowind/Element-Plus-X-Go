## GO-Element-Plus-X-Demo


## 快速开始

#### 环境准备（前端node环境搭建跳过）

1. 安装go 环境 https://golang.google.cn/
2. 安装wails https://wails.io/zh-Hans/docs/gettingstarted/installation


#### frontend 前端项目

按照正常的前端项目流程进行即可

#### wails

wails dev 启动
wails build 打包
wails build -nsis 打包为nsis安装包
wails build -nsis  -debug 打包为nsis安装包并打debug包
